Trilingual Spam Detection System
A secure, web-based Spam Detection System designed to analyze text content for spam patterns across three languages: English, Hindi, and Marathi. This system provides a centralized Admin Dashboard for managing the spam keyword database and a user-friendly workspace for real-time text analysis.

🚀 Features
Trilingual Support: Accurate detection for English, Hindi (Devanagari), and Marathi content.

Dynamic Intelligence: Admin panel allows for real-time addition and deletion of "Red Flag" keywords.

Secure Authentication: Role-based access control (Admin vs. User) with encrypted password handling.

Intuitive UI: Clean, modern interface designed for quick analysis.

🛠 Tech Stack
Backend: Java (Servlets, JDBC)

Frontend: HTML5, CSS3 (Glassmorphism design), JSP

Database: MySQL

Development Tools: Apache Tomcat, Maven

📋 Testing Overview
The system has undergone rigorous unit, integration, and system-level testing to ensure reliability and performance. Key test scenarios include:

Authentication: Validating secure registration and login workflows.

Core Detection: Verifying accurate pattern matching for multi-lingual input strings.

System Integrity: Handling concurrent user requests and database updates.

Clone the repository: git clone https://github.com/rudrakakde04-tech/Spam-Detection-System.git
